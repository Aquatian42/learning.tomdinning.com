The Cascade

Inheritance

Some styles are inherited by child elements. For example, all of the text in this div is the same font. Typographic sytles are usually inherited while others usually aren't.

Specificity

In cases where two conflicting styles apply to one element; more specific rules are applied over less specific rules. More selectors will also take precedence over fewer selectors.

  1. ID selectors
  2. Class selectors
  3. Type selectors
  4. Other selectors (like * - which has no specificity )

Any level of specificity will override inheritance

In the case of a conflict of specificity after all of these rules, the rule that was defined last in the css file (or the HTML file) witt be applied.
