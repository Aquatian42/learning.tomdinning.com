Arrays and Loops
Arrays are lists of items that are stored to variables. They can be declared using: const name = [item1, item2, item3] It is common to declare arrays as constants not variables in javascript. You can space out the items with as many spaces or line breaks as wanted. Arrays are dynamic and mutable in javascript, so they can be changes and the length can be changed after declaration, even if they are declared as constants. One array can contain multiple data types, even other arrays or objects!
Array elements can be accessed using indicies, myArray[0] would return the first element of myArray. This allows you to loop over them. You can also change specific indicies using myArray[index] = newValue
There are many built in methods for arrays in javascript. length is a property of arrays and returns the length of that array. THe last element of an array can beaccessed by getting it's length and subtracting one for the index. .push(element) is used to add elements to the end of an array. likewise, .pop() removes the last element from an array. .shift() removes the first element and .unshift(item) adds the specified item to the beginning of the array indexOf(element) will return the first index of that item in an array. join(seperator) will join all of the items in an array into a single string, seperated by whatever you have specified.
The most common types of loops are for and while loops, these can do the same things as each other but are formatted differently. They both run over and over until a certian condition is met. There are examples in this div's script element.
Arrays can be used to loop over the items of an array using for variableName of arrayName
Maps can be used to return an array that has had a particular function applied to every item in that array. Filters are similar but are passed a function which returns a boolean, the new array contains only items for which that boolean was true.
