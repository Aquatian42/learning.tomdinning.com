Data Types and Conditionals
There are 8 basic types of data in JavaScript. These can all be stored in variables without needing to declare the type making JavaScript a dynamically-typed language.
Numbers represent any numeric value, there is no distinction between Floats and Ints. Alongside these, there are the "special numberic valuse" NaN, Infinity and -Infintiy. Numbers can only be safely used up to 253 - 1 because they are stored as 64 bit floating point numbers with an 11 bit exponent and a 53 bit mantissa.
Numbers larger than this can be stored precicely using BigInt. This can be done by appending n to the end of the number. Eg: 12345678900987654321n or using BigInt() eg: BigInt(12345678900987654321). Operations involgving BigInts always return BigInts so, if nececary, they can be converted using Number()
Strings are strings of characters surrounded by quotes, they can be concatenated using the + operator. Single quotes (''), double quotes ("") or backticks (``) can be used. single and double quotes both declare normal strings. Backticks declare template literals, strings that can have variables embedded inside of them using ${} eg: `Hello ${name} you were born in {2025 - age}` They also respect line breaks so you don't need to use the \n character. Quotes con be included inside of strings by escaping the quote by putting a backslash before it eg: 'I\'m not going in there!' Strings are immutable.
Boolean data can only take the values true and false. They can be assigned directly or come as a result of comparisons.
null is a datatype of its own. It represents an unknown or empty value. It is usually assigned to variables that do not yet contain any data.
unassigned is similar to null in that it is the value given to variables with no data. It is different because it is just the default value such as when a variable is declared eg: let variable; variable would be unassigned.
Objects are used to store complex collections of data. All other datatypes are simple.
typeof element will return the type of the element.
Conditionals use if, else and else if. If and else if require brackets around the condition that they operate under. These all require curly braces to enclose the code that will execute if the outcome is true.
Logical operators operators that return true or false. JavaScript uses "||" for OR, "!" for NOT and "&&" for AND (Typed ALT + 038).
The Ternary Operator can be used instead of an if else statement. variable == true ? variable = false : variable = true
