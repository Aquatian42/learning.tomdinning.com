JavaScript basics
Functions are blocks of code that have been assigned a name, so only the name needs to be called to run that block. Functions can be defined in one place on a webpage and called elsewhere. The function in this block is defined in javascript.js which is called in the head. Parameters are the data that we declare must be passed to a function, and arguments are the real data that we pass into each instance of a function.

Functions are declared like this:

function new_fuinction(parameter1,parameter2) {

do things };

And called with new_function(argument1,argument2);

default values can be assigned to parameters using new_function(parameter1 = default_value,); when calling this function, the argument for parameter1 can be left empty and will default to default_value.
anonymous functions can be created without names by going function (parameters) These can be created and passed into other functions in one go.
scope is very importaant for functions in JavaScript. Variables, functions etc defined in a function are local to that function and can't be accessed from elsewhere in the code. Variables can only be declared once son if importing multiple js files, it is important ot have different vartiale names. Functions, however, can be redeclared and will be overwritten. Keeping variables in functions can help to mitigate these issues. Functions can access variables that have been declared outside of them, higher up. These are called global variables if they are outside of all functions.
