JavaScript basics
This div contains a script that sets a name and surname variable and then prints them to the console. It then changes the variables and prints them again.
Constants can be declared in javascript using const. These are like variables but are immutable. Trying to reassign a constant will result in an error.
Javascript uses (), /, *, +, -, **, %, ==, <, >, != in the same way as python.
JavaScript will try to convert strings that are just numbers into numbers for most numeric operations. The exception is the + operator because that is also the concatenation operator. NaN is a number type which indicates that a number is not a legal number such as when dividing a number by a string. Numbers can be converted into different bases using number.toString(base) eg: 10.toString(2) = 1010. == checks if two numbers are equal === checks if they are strictly equal (They have the same type and value.) 3=="3" True 3==="3" False.
