DOM Manipulation and Events
The document object model shows how elements on a HTML webpage related to each other. In Javascript, you can use CSS-style selectors or relationship properties to refrence desired nodes in the DOM. Relational selectors include firstElementChild and lastElementChild.
element.querySelector(selectors) will return a NodeList containing refrences to every match for the selectors, unlike element.querySelector(selector) which will return only the first match for the given selector. This NodeList can be converted to an array if required, using Array.from(desiredNodeList).
Element objects can also be created with document.createElement(type); these still need to be added to the dom using aNode.appendChild(childNode) or aNode.insertBefore(newNode, referenceNode) They can be removed using aNode.removeChild(childNode).
More than just the text can be changed, using .style or .setAttrribute lets you add any CSS styles to your node. There is also getAttribute and removeAttribute. Classes can be edited with .classList
